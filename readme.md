4. test： python3.5 test_models.py ucf101 RGB ../../action_dataset/action5.0/test_opencv_flow/labels_txt action5.0__opencv_rgb_model_best.pth.tar --arch BNInception --save_scores action5.0_test_rgb.npz --gpu 1 -j 1
